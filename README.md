# ING.-MECATRONICA
Proyecto Arduino: LED parpadeante
Este proyecto enciende y apaga un LED cada 1 segundo usando una placa Arduino.

Materiales
1 Arduino Uno
1 LED
1 resistencia de 220 ohmios
Cables puente
Protoboard
Conexión
Conecte el año del LED al pin 13.
Conecte el cátodo a GND usando la resistencia.
Archivos del proyecto
MiProyecto.ino
README.md
Cómo usarlo
Abra el archivo .inoen Arduino IDE.
Conecta tu placa Arduino al ordenador.
Sube el código a la placa.
Observe cómo parpadea el LED.
Código
El código principal está en el archivo .ino.
